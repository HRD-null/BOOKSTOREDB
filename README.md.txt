# BookStoreDB

Proyecto de base de datos transaccional para la gestión de ventas de libros usando MySQL.

## Contenido

- 01_create_database.sql – Crea la base de datos.
- 02_create_tables.sql – Define las tablas con relaciones.
- 03_fill_data.sql – Carga datos de ejemplo (clientes, libros, pedidos).
- 04_trigger.sql – Disminuye/restaura inventario automáticamente.
- 05_store_procedures.sql – Crea pedidos con múltiples líneas.
- 06_functions.sql – Calcula el total gastado por cliente.
- 07_create_view.sql –  Vista resumen de pedidos.
- 08_test_sp.sql – Consultas para pruebas.

## Requisitos

- MySQL 8.0+
- MySQL Workbench o cliente compatible
