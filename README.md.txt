# BookStoreDB

Proyecto de base de datos transaccional para la gestión de ventas de libros usando MySQL.

## Contenido

- 01_create_database.sql – Crea la base de datos.
- 02_create_tables.sql – Define las tablas con relaciones.
- 03_fill_data.sql – Carga datos de ejemplo (clientes, libros, pedidos).

## Requisitos

- MySQL 8.0+
- MySQL Workbench o cliente compatible
